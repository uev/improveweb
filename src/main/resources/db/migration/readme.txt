Принцип работы миграций flyway.
mvn flyway:baseline - состояние базы данных - первая версия
mvn flyway:info - просмотр информации об истории изменений базы данных
mvn flyway:validate - проверка схемы данныз
Именования миграция: если мажорная версия младше, то идём к минорной версии, иначе логика обратная (выполняется откат предыдущей транзакции)
